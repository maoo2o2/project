# BGM ファイルの置き場所

ここに以下のファイル名で MP3 を配置してください：

| ファイル名 | カテゴリ | 用途 |
|---|---|---|
| `upbeat1.mp3` | Upbeat | Happy Vlog ① |
| `upbeat2.mp3` | Upbeat | Happy Vlog ② |
| `chill1.mp3` | Chill | Chill Day ① |
| `chill2.mp3` | Chill | Chill Day ② |

## 推奨音源サイト

**Pixabay Music** — https://pixabay.com/music/
- 完全無料・商用利用OK・帰属表示不要
- MP3 でダウンロードしてそのまま配置可能

## 検索キーワード例

- Upbeat: `happy upbeat vlog`, `fun bright`
- Chill: `chill lofi`, `calm soft background`
